# Car Price Prediction using Decision Tree Regression

This project uses a Decision Tree Regression model to predict car prices based on various vehicle attributes.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Model
- Decision Tree Regression

## Workflow
1. Data Loading
2. Data Preprocessing
3. Exploratory Data Analysis
4. Train-Test Split
5. Model Training
6. Price Prediction
7. Model Evaluation

## Features
- Vehicle Age
- Present Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission
- Owner History

## Output
Predicts the estimated market price of a car.
